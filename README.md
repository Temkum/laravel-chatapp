# Laravel Post App

There's a lot covered in this project:

-   Authentication, 
-   laravel-mix to pull in a framework like tailwind or bootstrap,
-   Controllers
-   Routing, route model binding
-   database and migrations
-   models, soft deleting,
-   string helpers
-   blade templates and components
-   blade directives
-   eloquent (ORM)
-   model factories
-   authorisation
-   validation
-   sending email
-   eager loading and performance of SQL queries
-   pagination
-   redirection
-   CSRF protection
-   Form method spoofing
-   middleware
-   artisan console
-   dates and time
-   laravel collections
